## Singapore Business Intelligence Cheat Sheet

### Income Brackets by Store Type

| Indicator | Bracket |
|---|---|
| Cold Storage / Jason's Deli / Market Place | Premium (expat / high-income) |
| NTUC FairPrice | Mass-market (heartland HDB) |
| Sheng Siong | Working-class / budget |
| Guardian / Watsons | Middle income + female-skewing |
| Unity (NTUC Health) | Healthcare-focused, aging-adjacent |

### Hawker Centre Types

| Type | Significance |
|---|---|
| NEA-managed hawker centre | Public, subsidized, anchor for local dining culture |
| Kopitiam (branded) | Commercial operator (Kopitiam, Koufu, Foodfare), mid-tier |
| Private food court (e.g., Food Republic) | Premium positioning, mall-anchored |
| Void deck / coffeeshop | HDB residential base, captive commuter breakfast trade |

### Zoning Quick Reference

| Zone | Description | Business Fit |
|---|---|---|
| B1 / B2 | Industrial (light/heavy) | Logistics, F&B supply, not retail |
| Commercial | Shops, offices, F&B, hotels | All retail and hospitality |
| Residential (HDB) | Public housing, void decks | Community-serving retail, healthcare |
| White site | Mixed-use flexible | High-value development, premium retail |

### OneMap API (free geocoding)

```
https://www.onemap.gov.sg/api/common/elastic/search?searchVal=Tampines+MRT&returnGeom=Y&getAddrDetails=Y
```
No auth required. Returns lat/lng for Singapore addresses and MRT station names.
